<h1>IoChat</h1>

This project is the server part of the IoChat application.

<h3>Technology stack</h3>
<ul>
  <li>ASP.NET Core (8.0)</li>
  <li>Entity Framework (PostgreSQL)</li>
  <li>SignalR</li>
  <li>MediatR</li>
  <li>JWT</li>
  <li>ImageSharp</li>
</ul>

<h3>Installing</h3>

```git
git init
git clone https://github.com/EONCreator/IoChatServer.git
```
